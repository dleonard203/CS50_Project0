VS Code...

Use it!

VS Code is one of the best, free text editors out there. The web page you're on was built with VS Code. It allows users to:

  • Open entire folders in a side pane
  • Use intelliscence for code completion
  • Install plugins for boilerplate code
  • Access to a terminal, with the ability to spawn multiple
  • Git tracking and file highlighting to see which files have changed
  • Streamline programming
  • Quick-find function definitions with the right plug-ins (that are probably free)

You can download VS Code here and make your life more enjoyable
