What is Bash?

Bash stands for (B)ourne (a)gain (sh)ell. It is the free version of the Bourne shell, and comes pre-packaged with newer Macs

What can I do with Bash?

Scripting in Bash gives you access to low-level operating system behavior. You can:
  1. Move files
  2. Make files
  3. Connect to another machine securely through SSH
  4. Compile C code
  5. Gain insight into CPU and Memory usage through the top command
  6. Search all processes running with ps and search through them with grep
